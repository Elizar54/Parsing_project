Программа solution.py является парсером: она запрашивает текст поискового запроса пользователя, который тот вводит 
с клавиатуры. Этот запрос передается ресурсу lamoda.ru, где парсер добывает данные обо всех товарах по данному 
запросу. На выходе программа создает в папке с проектом excel-таблицу lamoda.xlsx, где содержатся все добытые данные.

Ограничения парсера: процесс парсинга довольно времязатратный - так, при хорошем интернет-соединении на сбор данных 
по запросу "кроссовки puma" (примерно 400 товаров) уходит около 3 минут, а по запросу "reebok" (примерно 1200 товаров) - 
около 9 минут. 
Отсюда рекомендуются следующие ограничения:
1) оптимальный запрос - не более чем на 2500 товаров (t <= 20 минут)
2) разумно-допустимый запрос - не более чем на 10000 товаров (t <= 1,5 часа)
Перед основными вычислительными действиями программа выводит количество найденных товаров по запросу, поэтому в случае
превышения названных цифр рекомендуется прекратить выполнение программы вручную.

Структура таблицы:
1) лист "Полные данные" - сюда входят все товары, для чьих карточек парсинг данных прошел успешно. По наблюдениям, таковых
около 98% от общего числа (например, при парсинге запроса на 266 товаров в этот лист вошли 260 из них; при парсинге запроса 
на 1273 товара в этот лист вошли 1260 из них). Товары по введенному запросу, которых нет в наличии, расположены в нижней части 
этого листа.
2) лист "Неполные данные" - сюда входят те товары, для чьих карточек парсинг данных не смог получить основные данные - таковых
около 2% от общего числа. Для таких товаров в таблицу выводятся ссылки на их карточки для возможности ручного заполнения. Для 
тех количеств, на которых целесообразно использовать парсер (<=2500, <=10000) число строк таблицы с "неполными данными" будет
относительно невелико (<=50, <=200), поэтому в случае необходимости получения всех данных рекомендуется ручное заполнение.

Заголовки таблицы:
vendor_code - артикул товара
brand - бренд товара
model - модель товара
price - действующая цена товара (если есть скидка, то она включена), руб.
discount - размер скидки, %
country - страна производства
link - ссылка на карточку товара (только для листа "Неполные данные")
